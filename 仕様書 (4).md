# シニアのための資産運用診断サイト 仕様書

## 1. プロジェクト概要

### 1.1 サービス名
**シニアのための資産運用診断**

### 1.2 目的
- シニア層が自身に適した資産運用スタイルを簡単に診断できるサービスを提供
- 診断結果をもとにLINE公式アカウントへ誘導し、個別相談につなげる

### 1.3 ターゲットユーザー
- **年齢層**: 50代後半〜70代
- **デバイス**: スマートフォン（メイン）、タブレット
- **特徴**: 
  - 退職金や年金の運用に関心がある
  - 投資経験は少ない〜中程度
  - デジタルリテラシーは中程度

---

## 2. サイト構成

### 2.1 ページ構成

```
[トップページ] → [診断ページ（質問1〜5）] → [結果ページ] → [LINE誘導]
```

| ページ | 内容 |
|--------|------|
| トップページ | サービス紹介、診断開始ボタン |
| 診断ページ | 質問の表示・回答選択（全5問） |
| 結果ページ | 診断結果表示、LINE誘導CTA |

---

## 3. 機能要件

### 3.1 トップページ

#### 表示要素
- サービスタイトル
- キャッチコピー（例：「あなたに合った資産運用を見つけましょう」）
- 診断の特徴説明（所要時間：約1分、全5問）
- **「診断をはじめる」ボタン**（大きく目立つデザイン）
- 信頼性を示す要素（監修情報など）

#### 機能
- 診断開始ボタンタップで診断ページへ遷移

---

### 3.2 診断ページ

#### 表示要素
- 進捗インジケーター（例：「質問 2 / 5」）
- プログレスバー
- 質問文
- 選択肢ボタン（3〜4択）
- 「戻る」ボタン（前の質問へ）

#### 機能
- 選択肢タップで次の質問へ自動遷移
- 前の質問に戻って回答変更可能
- 全問回答後、結果ページへ遷移

---

### 3.3 結果ページ

#### 表示要素

**上部：診断結果セクション**
- 診断タイプアイコン
- 診断タイプ名（例：「安定重視型」）
- タイプの説明文（100〜150文字程度）
- おすすめの運用方針
- 注意点やアドバイス

**下部：LINE誘導セクション**
- 区切り線またはカード形式で視覚的に分離
- 見出し：「あなたに合った資産運用のご相談はこちらから」
- 説明文：「専門スタッフが無料でご相談をお受けします」
- **LINE友だち追加ボタン**（緑色、LINEアイコン付き）
- 補足テキスト（例：「友だち追加後、診断結果をお送りください」）

**フッター**
- 「もう一度診断する」リンク
- 注意事項・免責事項

## 5. 診断ロジック

### 5.1 スコア計算
各質問の回答スコア（1〜4点）を合計する。

**合計スコア範囲**: 5点〜20点

### 5.2 診断タイプ判定

| スコア | 診断タイプ |
|--------|-----------|
| 5〜8点 | 安定重視型 |
| 9〜12点 | バランス型 |
| 13〜16点 | 成長志向型 |
| 17〜20点 | 積極運用型 |

---

## 6. 診断結果詳細

### 6.1 安定重視型

**アイコン**: 🛡️ 盾

**タイトル**: 安定重視型

**説明文**:
> あなたは資産の安全性を最も重視するタイプです。元本の保全を第一に考え、大きなリスクを取らない堅実な運用が向いています。

**おすすめの運用方針**:
- 元本確保型の商品を中心に
- 定期預金、個人向け国債など
- リスク資産は全体の10〜20%程度

**アドバイス**:
> インフレによる資産価値の目減りにも注意が必要です。専門家と一緒に、安全性と収益性のバランスを考えましょう。

---

### 6.2 バランス型

**アイコン**: ⚖️ 天秤

**タイトル**: バランス型

**説明文**:
> あなたは安定性と成長性のバランスを重視するタイプです。リスクを抑えながらも、ある程度の資産成長を目指す運用が向いています。

**おすすめの運用方針**:
- 国内外の債券と株式をバランスよく
- バランス型投資信託の活用
- リスク資産は全体の30〜50%程度

**アドバイス**:
> 定期的な見直しで、ライフステージに合わせた配分調整が大切です。プロのアドバイスを受けることをおすすめします。

---

### 6.3 成長志向型

**アイコン**: 📈 上昇グラフ

**タイトル**: 成長志向型

**説明文**:
> あなたは中長期的な資産の成長を重視するタイプです。ある程度のリスクを許容しながら、積極的なリターンを目指す運用が向いています。

**おすすめの運用方針**:
- 株式を中心としたポートフォリオ
- 国内外の株式投資信託
- リスク資産は全体の50〜70%程度

**アドバイス**:
> 短期的な価格変動に一喜一憂せず、長期的な視点で運用することが成功の鍵です。

---

### 6.4 積極運用型

**アイコン**: 🚀 ロケット

**タイトル**: 積極運用型

**説明文**:
> あなたは高いリターンを積極的に追求するタイプです。リスクを理解した上で、成長性の高い資産への投資が向いています。

**おすすめの運用方針**:
- 国内外の株式中心
- 成長株や新興国への投資も視野に
- リスク資産は全体の70%以上

**アドバイス**:
> 高いリターンには高いリスクが伴います。分散投資と定期的なリバランスを心がけましょう。

## 7. UI/UXデザイン要件

### 7.1 シニア向け配慮

| 項目 | 仕様 |
|------|------|
| 文字サイズ | 本文: 18px以上、見出し: 24px以上 |
| 行間 | 1.8〜2.0 |
| 色のコントラスト | WCAG AA基準以上（4.5:1以上） |
| ボタンサイズ | 最小48px × 48px、推奨56px以上 |
| タップ領域 | 十分な余白を確保 |
| フォント | ゴシック体（游ゴシック、Noto Sans JP） |

### 7.2 スマートフォン最適化

| 項目 | 仕様 |
|------|------|
| レイアウト | シングルカラム |
| ビューポート | 100vw、横スクロール禁止 |
| ボタン配置 | 画面下部に固定（thumb zone考慮） |
| 入力方式 | タップのみ（テキスト入力なし） |
| ローディング | スケルトンスクリーンまたはスピナー表示 |

### 7.3 カラースキーム

| 用途 | カラーコード | 説明 |
|------|-------------|------|
| メインカラー | #2E7D32 | 深緑（信頼感・安心感） |

| テキスト | #333333 | ダークグレー |

## 8. LINE誘導設計

### 8.1 CTA（Call to Action）設計

```
┌─────────────────────────────────────┐
│                                     │
│   あなたに合った資産運用の         │
│   ご相談はこちらから               │
│                                     │
│   専門スタッフが無料で             │
│   ご相談をお受けします             │
│                                     │
│  ┌─────────────────────────────┐   │
│  │  [LINE] LINEで相談する      │   │
│  └─────────────────────────────┘   │
│                                     │
│   ※ 友だち追加後、「診断結果」と   │
│     メッセージをお送りください     │
│                                     │
└─────────────────────────────────────┘
```

### 8.2 LINE友だち追加ボタン

- **遷移先**: LINE公式アカウントの友だち追加URL
- **形式**: `https://lin.ee/XXXXXXX` または `https://line.me/R/ti/p/@XXXXXXX`
- **ボタンデザイン**: 
  - 背景色: #06C755（LINEグリーン）
  - 文字色: #FFFFFF
  - LINEアイコン付き
  - 角丸: 8px
  - 影: 軽いドロップシャドウ

### 8.3 トラッキング

- UTMパラメータを付与して流入元を計測
- 例: `?utm_source=shindan&utm_medium=web&utm_campaign=senior_asset`
